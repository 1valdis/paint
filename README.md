# Paint [WiP]

This is my replica of MS Paint on canvas and [React](https://reactjs.org/). Created using [create-react-app](https://github.com/facebook/create-react-app).

### What can this thing do?
* [ ] Draw
  * [x] Pick colors
  * [x] Dropper
  * [x] Pen
  * [ ] Brushes
  * [x] Fill
  * [ ] Shapes
  * [ ] Text
* [ ] Edit
  * [x] Open file form disk
  * [x] Resize
  * [x] Selection
  * [x] Paste picture
  * [ ] Copy picture
  * [ ] Cut picture
  * [x] Save file to disk
